A small GUI demo for Slick, includes source and compiled classes. 

!!! NOTE: Oxygui is buggy and not actively developed. !!!

The oxy-gui.jar contains all the necessary GUI components and a default Aqua skin. (signed jar)
The oxy-test.jar contains all the test classes and test data. (signed jar)
The swing-layout-1.0.jar is needed to run the editor.
